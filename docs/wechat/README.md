# 公众号文章资源包

本目录包含发布到微信公众号所需的全部资源。

## 文件清单

| 文件 | 用途 |
|---|---|
| `article.html` | 完整 HTML 源码,可直接粘到公众号后台"HTML 模式"或第三方编辑器 |
| `home.png` | 主页截图(蓝色渐变 + 太阳 + 黄色车位线) |
| `settings.png` | 设置页截图(MiniMax / OpenAI / 通义千问三选一) |

| `about.png` | 关于页截图(Mr.Kon 创作者 + v0.5.0 版本号) |

## 发布步骤

### 方案 A:公众号后台(推荐)
1. 登录 https://mp.weixin.qq.com
2. 素材管理 → 新建图文消息
3. 切到"HTML"模式(部分编辑器叫"源代码")
4. 全文粘贴 `article.html` 的内容
5. 切回可视化模式,在 4 个 `<!-- IMAGE: xxx.png -->` 位置插入对应图片
6. 调整行距、字体大小 → 预览 → 群发

### 方案 B:第三方编辑器(135 编辑器、秀米等)
1. 把 `article.html` 导入第三方编辑器
2. 第三方会自动渲染基础样式 + 提示你替换图片
3. 4 张截图按 HTML 中标注的顺序插入(主页 → 设置 → 记录 → 关于)
4. 复制最终版粘贴到公众号
